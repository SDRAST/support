SSHTunnelManager¶

class support.trifeni.trifeni.util.SSHTunnelManager(logger=None)¶

Bases: object

Create and manage SSH tunnel connections.

Like individual SSHTunnel instances, this can, and should be used as a context manager.

Examples:

Automatically clean up any tunnels created.

with SSHTunnelManager() as manager:
    tunnel0 = manager.create_tunnel("remote_alias", "localhost", 9090, 9090)
    tunnel1 = manager.create_tunnel("remote_alias", "localhost", 9091, 9091)

cleanup()¶

Destroy all the tunnels associated with the manager. Note that this doesn’t actually delete the SSHTunnel objects from memory.
