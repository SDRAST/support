cleanup()¶

Destroy all the tunnels associated with the manager. Note that this doesn’t actually delete the SSHTunnel objects from memory.

find_nameserver(local_ns_port=None)¶

Find the remote nameserver. This gets called by __init__.

Examples:

If I have a nameserver running locally on port 9090, and there is a nameserver on the remote running on 9090, I can’t forward port 9090 to the remote! Enter the local_ns_port keyword argument:

tunnel = NameServerTunnel(remote_server_name="remote_alias", ns_port=9090)
ns = tunnel.find_nameserver(local_ns_port=9091)
Parameters

local_ns_port (int, optional) – Use a local forwarding port that is different from self.ns_port.

Returns

Pyro4.naming.NameServer
