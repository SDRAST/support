LAN_hosts_status¶

support.network.LAN_hosts_status(port=22, private=False)¶

Get information about and status of local network hosts with port open

Returns a dict with::

  • “up”: hosts up

  • “down”: hosts down

  • “IP”: IP addresses

  • “MAC”: MAC addresses

  • “ROACH”: list of ROACHes

This is too crude; very senstive to format changes
