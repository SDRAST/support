Process¶

class support.trifeni.trifeni.util.Process(name='', pid=0, ps_line=None, command_name='ssh')¶

Bases: object

Class representing a basic process, with name and pid. A Process object is used to represent an already running process. To spawn a new process, use subprocess.subprocess.Popen.
